# Product Roadmap

This roadmap describes the capability-based evolution of AAES-OS.

## Milestones

- v1.0: stable core platform
- v1.1: agent intelligence
- v1.2: simulation platform
- v2.0: enterprise platform
- v3.0: federated intelligence
- v4.0: advanced research platform

## Release rule

A release is complete only when architecture, APIs, docs, tests, governance, and a reference implementation are aligned.
