# AAIS

AAIS is the governed immune layer, simplified to a three-step path:

`llm -> jarvis -> nova`

- `llm` normalizes the request
- `jarvis` applies doctrine and invariants
- `nova` dispatches the approved check onto the bus

AAIS now also exposes the capability catalog behind the surrounding surface, including the reference runtime composer, conformance suite generator, and implementation gap resolver.

See the live catalog page: [AAIS Capability Registry](./aais-capabilities.md)
