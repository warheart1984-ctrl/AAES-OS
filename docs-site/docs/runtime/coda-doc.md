# CodaDoc

CodaDoc is the live documentation catalog surface for the Coda stack.
It enumerates the corpus records that separate live package surfaces from doc-forward corpus families.

## Package source

- [packages/coda-doc/src/index.ts](../../../packages/coda-doc/src/index.ts)

## Runtime contract

- Build the Coda corpus catalog
- Classify live, mixed, and doc-forward surfaces
- Resolve corpus records by name or alias
- Provide a stable corpus summary for runtime and docs pages
