# Capability Graph

The capability graph defines explicit permissions for agents, backends, and runtime components.

## Integration Points

- **CodingRouter** — backend allowlists from policy routing rules
- **Architect Agent** — UCR span guards and output patches
- **Ops Console** — visualizes capability boundaries

## Future Work

A dedicated `@aaes-os/capability-graph-visualizer` package will render the graph interactively in the ops console.
