### Questions and Discussions

For questions about:
- Monorepo structure and workspaces
- Package development
- Build and test procedures
- Ops console usage
- Constitutional compliance
- UCDD standards implementation

Please open the repository Discussions page with the appropriate category.

### Bug Reports

If you encounter a bug, please:
1. Check existing repository Issues
2. If not already reported, open a new issue using the bug report template
3. Include environment details, reproduction steps, and expected behavior
4. Specify which package is affected
5. **Include traceability links** per UCDD S-002 (constitutional clauses, UCDD standards)

### Feature Requests

We welcome feature requests! Please:
1. Check existing repository Issues
2. Open a new issue using the feature request template
3. Describe the problem, proposed solution, and alternatives considered
4. Specify which package the feature is for
5. **Include traceability links** per UCDD S-002
