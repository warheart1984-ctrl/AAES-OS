# Twin Civilization Map

Cluster: `sovereign-twin-cluster-01`

Kernel companion: [CIEMS Kernel Architecture](./CIEMS_KERNEL_ARCHITECTURE.md).

The civilization map is a composition of constitutional projections over the frozen USS graphs and CIEMS governance records. `SovereigntyGraph`, `RoleGraph`, `ConsensusGraph`, and `FederationGraph` are logical projections—not additional physical graph engines. `SimulationGraph` is an isolated state target. The physical USS remains Identity, Evidence, Reputation, Mission, Temporal, Lineage, and Multiverse.

| Projection | Authoritative backing | Principal relationship |
|---|---|---|
| Identity | IdentityGraph | Twin identity, principal delegation, role and personality references |
| Sovereignty | IdentityGraph + CIEMS contracts | Twin to allowed scopes, graphs, and capabilities |
| Role | IdentityGraph + role contracts | Role lattice and risk ceiling |
| Mission | MissionGraph + EvidenceGraph | Intent to plan to execution to evidence |
| Consensus | EvidenceGraph + LineageGraph | Proposal, votes, result, and derived profile |
| Multiverse | MultiverseGraph | Reality branches and trajectories |
| Simulation | isolated SimulationGraph | Simulated intent, plan, evidence, and outcome |
| Federation | EvidenceGraph + LineageGraph | Origin, evaluation, admission, and local import |
| Lineage | LineageGraph + TemporalGraph | Immutable constitutional sequence and replay |
| Evidence | EvidenceGraph | Provenance web binding actors, missions, governance, and substrates |

Every projected node and edge carries an evidence or contract reference, realm where applicable, governance profile, validity interval, and update identifier. Replay reconstructs a projection from immutable updates rather than mutating historical records.

The five-Twin layout places the Strategic Twin above the cluster, Execution Twin below, Negotiation Twin at the center, Simulation Twin to the left, and Federation Twin to the right. Weighted consensus edges encode strategist 3, negotiator 3, executor 2, simulator 1, and federator 3. Federation admission and constitutional amendment paths remain unanimous.
