## 1. Freeze declaration

The Implementation Profile Freeze Criteria are hereby designated **Frozen** and constitute the governing freeze standard for `profile-*` artifacts.

These criteria define the minimum evidence required for a profile to graduate from `Draft` to `Frozen`.

## 3. Constitutional meaning

The criteria remain subordinate to CIS Core v1.0.

They establish when an implementation profile may be considered a frozen companion baseline without weakening the constitutional layer.

## 4. Change control

Any future change to these criteria must proceed through the governed constitutional change process.

No implementation detail or informal agreement may alter the frozen freeze-criteria baseline.
