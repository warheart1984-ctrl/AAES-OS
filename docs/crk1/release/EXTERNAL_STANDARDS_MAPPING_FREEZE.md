# External Standards Mapping Layer Freeze Notice

**Release family:** CRK-1 companion specification  
**Status:** Frozen  
**Effective date:** 2026-07-12  
**Scope:** External Standards Mapping Layer

---

## 1. Freeze declaration

The External Standards Mapping Layer is hereby designated **Frozen** and constitutes a frozen companion baseline for the CIS ecosystem.

This freeze establishes the standards-mapping surface for ISO, NIST, IEEE, W3C, and IETF as a stable companion artifact.

## 3. Constitutional meaning

The external standards mapping layer remains subordinate to CIS Core v1.0.

It shall map CIS obligations to external standards families without changing constitutional terminology or requirements.

## 4. Change control

Any future change to the external standards mapping terminology or family mappings must proceed through the governed constitutional change process.

No implementation detail or informal agreement may alter the frozen external-standards baseline.
