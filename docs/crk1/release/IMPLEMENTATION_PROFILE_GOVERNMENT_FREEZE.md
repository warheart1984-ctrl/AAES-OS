The Government profile remains subordinate to CIS Core v1.0.

It may specialize public-sector adaptation patterns, but it may not redefine governance, evidence, replay, or constitutional terminology.

## 4. Change control

Any future change to this profile must proceed through the governed constitutional change process and satisfy the Implementation Profile Freeze Criteria.

No implementation detail or informal agreement may alter the frozen Government profile baseline.
